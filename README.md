NestJS
MySQL

## How To Use

### Install

```
$ npm install
```

### Config .env

```
PORT=/* Backend port number to use */
DB_HOST=/* DB address (localhost or AWS RDS) */
DB_USERNAME=/* DB admin name */
DB_PASSWORD=/* DB admin password */
DB_DATABASE=/* DB name to use */
COOKIE_SECRET=/* secret key */
```

### Running the app

```
$ npm run start:dev
```

### Create DB (Default: MySQL)

```
$ npm run db:create
```

### Create Table

```
$ npm run schema:sync
```
