# Project Community

개발 중..

## How To Use

### Install

```
$ npm insall
```

### Config .env

```
PORT=/* 사용할 백엔드 포트 번호 */
DB_HOST=/* aws rds 주소 */
DB_USERNAME=/* aws rds admin 이름 */
DB_PASSWORD=/* aws rds admin 패스워드 */
DB_DATABASE=/* 사용할 데이터 베이스 이름 */
```

### Running the app

```
$ npm run start:dev
```
